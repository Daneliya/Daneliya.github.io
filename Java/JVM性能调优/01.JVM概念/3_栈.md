# 栈

## 栈的作用

栈内存，主管程序的运行，生命周期和线程同步；
线程结束，栈内存也就释放了，对于栈来说，**不存在垃圾回收问题**。

## 栈存储的东西

8大基本类型、对象引用，实例的方法。
