дёҖгҖҒжҗһжҮӮAPIзҡ„е…ій”®еҜ№иұЎ вҖӢ

йҷӨдәҶ LangChain жЎҶжһ¶пјҢOpenAI зҡ„ Assistant API д№ҹжҳҜжһ„е»әејәеӨ§ AI еҠ©жүӢзҡ„й«ҳж•Ҳе·Ҙе…·гҖӮдёҺ LangChain зӣёжҜ”пјҢдәҢиҖ…е®ҡдҪҚдёҺеҠҹиғҪеӯҳеңЁжҳҫи‘—е·®ејӮпјҢд»ҘдёӢе°Ҷи§Јжһҗ Assistant API зҡ„ж ёеҝғжҰӮеҝөгҖҒйҖӮз”ЁеңәжҷҜеҸҠдёҺ LangChain зҡ„еҢәеҲ«гҖӮ

1гҖҒAssistant API дёҺ LangChain зҡ„ж ёеҝғеҢәеҲ« вҖӢ

еҜ№жҜ”з»ҙеәҰOpenAI Assistant APILangChain
жң¬иҙЁеҹәдәҺ API зҡ„жңҚеҠЎпјҲзӣҙжҺҘи°ғз”Ё OpenAI жЁЎеһӢиғҪеҠӣпјүйҖҡз”Ё AI еә”з”ЁејҖеҸ‘жЎҶжһ¶пјҲж•ҙеҗҲеӨҡз§Қе·Ҙе…· / жЁЎеһӢпјү
жЁЎеһӢж”ҜжҢҒд»…ж”ҜжҢҒ OpenAI жЁЎеһӢпјҲеҰӮ GPT-4гҖҒGPT-3.5пјүж”ҜжҢҒеӨҡжқҘжәҗжЁЎеһӢпјҲOpenAIгҖҒAnthropicгҖҒжң¬ең°жЁЎеһӢзӯүпјү
ж ёеҝғзӣ®ж Үеҝ«йҖҹе®һзҺ°еҹәдәҺ OpenAI жЁЎеһӢзҡ„еҜ№иҜқдёҺе·Ҙе…·и°ғз”Ёжһ„е»әеӨҚжқӮ AI еә”з”ЁпјҲйӣҶжҲҗе·Ҙе…·й“ҫгҖҒи®°еҝҶгҖҒи·Ҝз”ұзӯүпјү

2гҖҒAssistant API зҡ„ж ёеҝғеҜ№иұЎ вҖӢ

зҗҶи§Ј Assistant API йңҖжҺҢжҸЎд»ҘдёӢе…ій”®еҜ№иұЎпјҢе®ғд»¬е…ұеҗҢжһ„жҲҗдәҶеҜ№иҜқдёҺдәӨдә’зҡ„еҹәзЎҖпјҡ

1.1. AssistantпјҲеҠ©жүӢпјү вҖӢ

  • е®ҡд№үпјҡе…·еӨҮзү№е®ҡиғҪеҠӣзҡ„ AI е®һдҪ“пјҢеҹәдәҺ OpenAI жЁЎеһӢжһ„е»әпјҢеҸҜи°ғз”Ёе·Ҙе…·гҖӮ
  • ж ёеҝғеҸӮж•°пјҡ
    • жЁЎеһӢпјҲеҰӮgpt-4пјүпјӣ
    • еҗҚз§°гҖҒжҸҸиҝ°пјҲж ҮиҜҶеҠ©жүӢеҠҹиғҪпјүпјӣ
    • зі»з»ҹжҢҮд»ӨпјҲжҢҮеҜјеҠ©жүӢиЎҢдёәзҡ„ promptпјүпјӣ
    • еҸҜз”Ёе·Ҙе…·пјҲд»Јз Ғи§ЈйҮҠеҷЁгҖҒжЈҖзҙўгҖҒиҮӘе®ҡд№үеҮҪж•°зӯүпјүгҖӮ

1.2. ThreadпјҲдјҡиҜқзәҝзЁӢпјү вҖӢ

  • е®ҡд№үпјҡеӯҳеӮЁз”ЁжҲ·дёҺеҠ©жүӢд№Ӣй—ҙзҡ„дёҖзі»еҲ—еҜ№иҜқпјҢзӣёеҪ“дәҺ вҖңеҜ№иҜқдёҠдёӢж–Үе®№еҷЁвҖқгҖӮ
  • еҠҹиғҪпјҡ
    • дҝқеӯҳж¶ҲжҒҜеҺҶеҸІпјҲз”ЁжҲ·жҸҗй—®гҖҒеҠ©жүӢеӣһеӨҚпјүпјӣ
    • иҮӘеҠЁеӨ„зҗҶдёҠдёӢж–Үй•ҝеәҰпјҡеҪ“еҜ№иҜқи¶…иҝҮжЁЎеһӢдёҠдёӢж–ҮзӘ—еҸЈж—¶пјҢиҮӘеҠЁжҲӘж–ӯж—©жңҹеҶ…е®№пјҲдҝқз•ҷе…ій”®дҝЎжҒҜпјүгҖӮ

1.3. MessageпјҲж¶ҲжҒҜпјү вҖӢ

  • е®ҡд№үпјҡз”ЁжҲ·жҲ–еҠ©жүӢеңЁеҜ№иҜқдёӯеҸ‘йҖҒзҡ„еҶ…е®№пјҢжҳҜ Thread зҡ„з»„жҲҗеҚ•е…ғгҖӮ
  • ж јејҸпјҡж”ҜжҢҒж–Үжң¬гҖҒеӣҫзүҮгҖҒж–ҮжЎЈзӯүеӨҡжЁЎжҖҒеҶ…е®№гҖӮ
  • еӯҳеӮЁпјҡд»ҘеҲ—иЎЁеҪўејҸе…іиҒ”еҲ° ThreadпјҢжҢүж—¶й—ҙйЎәеәҸжҺ’еҲ—гҖӮ

1.4. RunпјҲиҝҗиЎҢпјү вҖӢ

  • е®ҡд№үпјҡеңЁжҹҗдёӘ Thread дёҠи°ғз”Ё Assistant жү§иЎҢд»»еҠЎзҡ„еҠЁдҪңгҖӮ
  • иҝҮзЁӢпјҡ
    • еҠ©жүӢз»“еҗҲ Thread дёӯзҡ„еҺҶеҸІеҜ№иҜқпјҢи°ғз”ЁжЁЎеһӢе’Ңе·Ҙе…·пјӣ
    • з”ҹжҲҗж–°зҡ„еӣһеӨҚж¶ҲжҒҜпјҢе№¶ж·»еҠ еҲ° Thread дёӯпјҢе®һзҺ°иҝһз»ӯеҜ№иҜқгҖӮ

1.5. RunStepпјҲиҝҗиЎҢжӯҘйӘӨпјү вҖӢ

  • е®ҡд№үпјҡRun иҝҮзЁӢдёӯеҠ©жүӢжү§иЎҢзҡ„е…·дҪ“ж“ҚдҪңпјҲеҰӮи°ғз”Ёе·Ҙе…·гҖҒз”ҹжҲҗж–Үжң¬зӯүпјүгҖӮ
  • дҪңз”ЁпјҡеҸҜжҹҘзңӢеҠ©жүӢзҡ„ вҖңжҖқиҖғиҝҮзЁӢвҖқпјҢеё®еҠ©и°ғиҜ•жҲ–зҗҶи§Јз»“жһңз”ҹжҲҗйҖ»иҫ‘пјҲзұ»дјј LangChain зҡ„verboseжЁЎејҸпјүгҖӮ

3гҖҒAssistant API ж”ҜжҢҒзҡ„е·Ҙе…· вҖӢ

дёҺ LangChain зҡ„ Agent иғҪеҠӣзұ»дјјпјҢAssistant API ж”ҜжҢҒеӨҡз§Қе·Ҙе…·жү©еұ•еҠ©жүӢеҠҹиғҪпјҡ

  1. Code InterpreterпјҲд»Јз Ғи§ЈйҮҠеҷЁпјүпјҡиҝҗиЎҢ Python д»Јз ҒпјҢеӨ„зҗҶи®Ўз®—гҖҒж•°жҚ®еҸҜи§ҶеҢ–зӯүд»»еҠЎгҖӮ
  2. Knowledge RetrievalпјҲж–ҮжЎЈжЈҖзҙўпјүпјҡдёҠдј ж–ҮжЎЈпјҲеҰӮ PDFгҖҒTXTпјүпјҢи®©еҠ©жүӢеҹәдәҺж–ҮжЎЈеҶ…е®№еӣһзӯ”й—®йўҳпјҲзұ»дјј LangChain зҡ„еҗ‘йҮҸеӯҳеӮЁжЈҖзҙўпјүгҖӮ
  3. Function CallingпјҲиҮӘе®ҡд№үеҮҪж•°и°ғз”Ёпјүпјҡи°ғз”ЁеӨ–йғЁ API жҲ–е·Ҙе…·пјҢжү©еұ•еҠ©жүӢиғҪеҠӣпјҲеҰӮжҹҘиҜўеӨ©ж°”гҖҒж“ҚдҪңж•°жҚ®еә“пјүгҖӮ

4гҖҒйҖӮз”ЁеңәжҷҜ вҖӢ

  • еҝ«йҖҹејҖеҸ‘еҹәдәҺ OpenAI жЁЎеһӢзҡ„еҜ№иҜқеҠ©жүӢпјҲж— йңҖжҗӯе»әеӨҚжқӮжЎҶжһ¶пјүпјӣ
  • йңҖиҰҒеҲ©з”Ёд»Јз Ғи§ЈйҮҠеҷЁгҖҒж–ҮжЎЈжЈҖзҙўзӯүе·Ҙе…·зҡ„еңәжҷҜпјҲеҰӮж•°жҚ®еҲҶжһҗгҖҒж–ҮжЎЈй—®зӯ”пјүпјӣ
  • дё“жіЁдәҺ OpenAI з”ҹжҖҒпјҢж— йңҖе…је®№е…¶д»–жЁЎеһӢзҡ„йЎ№зӣ®гҖӮ

5гҖҒжҖ»з»“ вҖӢ

Assistant API жҳҜ OpenAI жҸҗдҫӣзҡ„ вҖңеҚіз”ЁеһӢвҖқ еҜ№иҜқжңҚеҠЎпјҢж ёеҝғдјҳеҠҝеңЁдәҺз®ҖеҢ–еҹәдәҺе…¶жЁЎеһӢзҡ„еҠ©жүӢејҖеҸ‘жөҒзЁӢпјҢйҖӮеҗҲеҝ«йҖҹиҗҪең°дё”дҫқиө– OpenAI жЁЎеһӢзҡ„еңәжҷҜгҖӮе…¶ж ёеҝғеҜ№иұЎпјҲAssistantгҖҒThreadгҖҒRun зӯүпјүжһ„жҲҗдәҶе®Ңж•ҙзҡ„еҜ№иҜқз”ҹе‘Ҫе‘ЁжңҹпјҢе·Ҙе…·иғҪеҠӣдёҺ LangChain зҡ„ Agent жңүзӣёйҖҡд№ӢеӨ„пјҢдҫҝдәҺе·Іжңү LangChain дҪҝз”Ёз»ҸйӘҢзҡ„ејҖеҸ‘иҖ…еҝ«йҖҹдёҠжүӢгҖӮ

дәҢгҖҒеҲӣе»әз§Ғдәәж•°еӯҰеҠ©жүӢ вҖӢ

еҖҹеҠ© OpenAI Assistant API зҡ„д»Јз Ғи§ЈйҮҠеҷЁеҠҹиғҪпјҢеҸҜеҝ«йҖҹжһ„е»әдёҖдёӘиғҪеҮҶзЎ®и§Јзӯ”ж•°еӯҰй—®йўҳзҡ„ AI еҠ©жүӢгҖӮд»ҘдёӢжҳҜе…·дҪ“е®һзҺ°жӯҘйӘӨпјҢеҢ…жӢ¬еҠ©жүӢеҲӣе»әгҖҒеҜ№иҜқз®ЎзҗҶеҸҠиҝҗиЎҢжөҒзЁӢпјҡ

2.2. еҲӣе»ә ThreadпјҲдјҡиҜқзәҝзЁӢпјү вҖӢ

жҜҸдёӘз”ЁжҲ·жҲ–еҜ№иҜқзӘ—еҸЈеҜ№еә”дёҖдёӘзӢ¬з«Ӣзҡ„ ThreadпјҢз”ЁдәҺеӯҳеӮЁеҜ№иҜқеҺҶеҸІгҖӮ

и°ғз”Ёthreads.runs.createи®©еҠ©жүӢеӨ„зҗҶ Thread дёӯзҡ„ж¶ҲжҒҜпјҢз”ҹжҲҗеӣһзӯ”гҖӮ

2.5. зӯүеҫ…иҝҗиЎҢе®ҢжҲҗе№¶иҺ·еҸ–з»“жһң вҖӢ

з”ұдәҺеҠ©жүӢз”ҹжҲҗеӣһзӯ”йңҖиҰҒж—¶й—ҙпјҢйңҖйҖҡиҝҮеҫӘзҺҜжҹҘиҜўиҝҗиЎҢзҠ¶жҖҒпјҢзӣҙиҮіе®ҢжҲҗгҖӮ

2.6. иҺ·еҸ–е№¶еұ•зӨәеҜ№иҜқеҺҶеҸІ вҖӢ

иҝҗиЎҢе®ҢжҲҗеҗҺпјҢйҖҡиҝҮthreads.messages.listиҺ·еҸ– Thread дёӯзҡ„жүҖжңүж¶ҲжҒҜпјҲз”ЁжҲ·жҸҗй—® + еҠ©жүӢеӣһзӯ”пјүгҖӮ

2.7. е°ҒиЈ…дәӨдә’еҮҪж•°пјҲз®ҖеҢ–еӨҡиҪ®еҜ№иҜқпјү вҖӢ

дёәж–№дҫҝеӨҡиҪ®жҸҗй—®пјҢеҸҜе°Ҷ вҖңж·»еҠ ж¶ҲжҒҜвҶ’иҝҗиЎҢеҠ©жүӢвҶ’иҺ·еҸ–з»“жһңвҖқ е°ҒиЈ…дёәеҮҪж•°гҖӮ

2.8. е®Ңж•ҙд»Јз ҒзӨәдҫӢ вҖӢ

# еҲқе§ӢеҢ–OpenAIе®ўжҲ·з«ҜпјҲйңҖжҸҗеүҚй…ҚзҪ®APIеҜҶй’ҘпјҢеҰӮзҺҜеўғеҸҳйҮҸпјү
client = OpenAI()

# step1пјҡеҲӣе»әдёҖдёӘж•°еӯҰеҠ©жүӢ

    model="gpt-3.5-turbo",  # еҹәдәҺGPT-3.5-turboжЁЎеһӢ
    name="ж•°еӯҰеҠ©жүӢ",  # еҠ©жүӢеҗҚз§°
    instructions="дҪ жҳҜдёҖдёӘж•°еӯҰеҠ©жүӢпјҢеҸҜд»ҘйҖҡиҝҮзј–еҶҷе’ҢиҝҗиЎҢд»Јз ҒжқҘеӣһзӯ”ж•°еӯҰзӣёе…ій—®йўҳгҖӮ",  # еҠ©жүӢзҡ„ж ёеҝғжҢҮд»Ө
    tools=[{"type": "code_interpreter"}]  # еҗҜз”Ёд»Јз Ғи§ЈйҮҠеҷЁе·Ҙе…·пјҲз”ЁдәҺи®Ўз®—ж•°еӯҰй—®йўҳпјү
)

    role="user",  # ж¶ҲжҒҜи§’иүІдёәз”ЁжҲ·
    content="жҲ‘йңҖиҰҒи§ЈиҝҷдёӘж–№зЁӢ`5x^2вҲ’1200x+72000=0пјҢжңӘзҹҘж•°еә”иҜҘжҳҜеӨҡе°‘пјҹ"  # е…·дҪ“й—®йўҳеҶ…е®№
)

# step4пјҡи§ҰеҸ‘еҠ©жүӢиҝҗиЎҢпјҢеӨ„зҗҶеҪ“еүҚзәҝзЁӢдёӯзҡ„ж¶ҲжҒҜ

)

run

    message = client.beta.threads.messages.create(
    thread_id=thread.id,
    role="user",
    content=prompt  # з”ЁжҲ·зҡ„ж–°й—®йўҳ
    )
    
    # 2. и§ҰеҸ‘еҠ©жүӢиҝҗиЎҢпјҢеӨ„зҗҶж–°ж¶ҲжҒҜ
    run = client.beta.threads.runs.create(
      thread_id=thread.id,
      assistant_id=assistant.id,
      instructions=run_instruction  # жң¬ж¬ЎиҝҗиЎҢзҡ„йўқеӨ–жҢҮд»ӨпјҲеҸҜйҖүпјү
    )
    
    # 3. зӯүеҫ…иҝҗиЎҢе®ҢжҲҗ
    while run.status != "completed":
        # е®һж—¶жҹҘиҜўиҝҗиЎҢзҠ¶жҖҒ
        keep_retrieving_run = client.beta.threads.runs.retrieve(
            thread_id=thread.id,
            run_id=run.id
        )
        print(f"Run status: {keep_retrieving_run.status}")  # жү“еҚ°зҠ¶жҖҒ
        
        # иӢҘе®ҢжҲҗпјҢйҖҖеҮәеҫӘзҺҜ
        if keep_retrieving_run.status == "completed":
            break
    
    # 4. иҺ·еҸ–е№¶жү“еҚ°жүҖжңүж¶ҲжҒҜпјҲеҢ…еҗ«еҺҶеҸІеҜ№иҜқе’Ңж–°еӣһзӯ”пјү
    messages = client.beta.threads.messages.list(
        thread_id=thread.id
    )
    
    for data in messages.data:
        print("\n")  # жҚўиЎҢеҲҶйҡ”
        print(data.content[0].text.value)  # жү“еҚ°ж¶ҲжҒҜеҶ…е®№
        print("------")  # еҲҶйҡ”зәҝ


# жөӢиҜ•еӨҡиҪ®еҜ№иҜқпјҡй—®дёҖдёӘж–°й—®йўҳпјҲ2зҡ„56ж¬Ўж–№жҳҜеӨҡе°‘пјү

    purpose="assistants"  # еЈ°жҳҺж–Үд»¶з”ЁйҖ”дёә"дҫӣеҠ©жүӢдҪҝз”Ё"пјҲеҝ…йЎ»и®ҫзҪ®жӯӨеҖјпјү

    instructions="дҪ жҳҜдёҖдёӘжҷәиғҪеҠ©жүӢпјҢеҸҜд»Ҙи®ҝй—®ж–Үд»¶жқҘеӣһзӯ”дәәе·ҘжҷәиғҪйўҶеҹҹи®әж–Үзҡ„зӣёе…ій—®йўҳгҖӮ",  # еҠ©жүӢзҡ„ж ёеҝғжҢҮд»ӨпјҲиҜҙжҳҺе…¶еҠҹиғҪпјү

        thread_id=thread.id,  # е…іиҒ”еҲ°еҪ“еүҚеҜ№иҜқзәҝзЁӢ
        role="user",  # ж¶ҲжҒҜи§’иүІдёә"з”ЁжҲ·"
        content=prompt  # з”ЁжҲ·зҡ„е…·дҪ“й—®йўҳ
    )
    
    # 2. и§ҰеҸ‘еҠ©жүӢиҝҗиЎҢпјҢеӨ„зҗҶеҪ“еүҚзәҝзЁӢдёӯзҡ„й—®йўҳ
    run = client.beta.threads.runs.create(
        thread_id=thread.id,  # зӣ®ж ҮеҜ№иҜқзәҝзЁӢID
        assistant_id=assistant.id,  # и°ғз”ЁеүҚйқўеҲӣе»әзҡ„и®әж–Үй—®зӯ”еҠ©жүӢ
        instructions=run_instruction  # жң¬ж¬ЎиҝҗиЎҢзҡ„йўқеӨ–жҢҮд»ӨпјҲеҸҜйҖүпјҢжӯӨеӨ„дёәз©әпјү
    )
    
    # 3. еҫӘзҺҜзӯүеҫ…еҠ©жүӢеӨ„зҗҶе®ҢжҲҗпјҲйҒҝе…ҚжҸҗеүҚиҺ·еҸ–жңӘз”ҹжҲҗзҡ„з»“жһңпјү
    while run.status != "completed":
        # е®һж—¶жҹҘиҜўиҝҗиЎҢзҠ¶жҖҒпјҲqueued/in_progress/completedзӯүпјү
        keep_retrieving_run = client.beta.threads.runs.retrieve(
            thread_id=thread.id,
            run_id=run.id
        )
        print(f"Run status: {keep_retrieving_run.status}")  # жү“еҚ°еҪ“еүҚзҠ¶жҖҒ
        
        # иӢҘиҝҗиЎҢе®ҢжҲҗпјҢйҖҖеҮәеҫӘзҺҜ
        if keep_retrieving_run.status == "completed":
            break
    
    # 4. иҺ·еҸ–е№¶жү“еҚ°еҜ№иҜқзәҝзЁӢдёӯзҡ„жүҖжңүж¶ҲжҒҜпјҲеҢ…еҗ«з”ЁжҲ·й—®йўҳе’ҢеҠ©жүӢеӣһзӯ”пјү
    messages = client.beta.threads.messages.list(
        thread_id=thread.id
    )
    
    # йҒҚеҺҶж¶ҲжҒҜеҲ—иЎЁпјҢжү“еҚ°жҜҸжқЎж¶ҲжҒҜеҶ…е®№
    for data in messages.data:
        print("\n")  # жҚўиЎҢеҲҶйҡ”дёҚеҗҢж¶ҲжҒҜ
        print(data.content[0].text.value)  # жҸҗеҸ–ж¶ҲжҒҜзҡ„ж–Үжң¬еҶ…е®№
        print("------")  # з”ЁеҲҶйҡ”зәҝеҢәеҲҶж¶ҲжҒҜ

# и°ғз”ЁеҮҪж•°еҗ‘еҠ©жүӢжҸҗй—®пјҡжҹҘиҜўд»Ӣз»ҚTransformerжһ¶жһ„зҡ„и®әж–ҮеҸҠй“ҫжҺҘ
get_response_from_assistant(assistant, thread, "е“ӘзҜҮи®әж–Үд»Ӣз»ҚдәҶTransformerжһ¶жһ„пјҹи®әж–Үй“ҫжҺҘжҳҜд»Җд№Ҳпјҹ")
